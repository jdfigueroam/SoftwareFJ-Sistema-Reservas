# SoftwareFJ-Sistema-Reservas

Proyecto desarrollado para la Fase 4 del curso de Programación (213023A_2201) de la Universidad Nacional Abierta y a Distancia – UNAD.

El sistema fue diseñado para la empresa Software FJ con el objetivo de gestionar clientes, servicios y reservas mediante Programación Orientada a Objetos en Python, sin utilizar bases de datos.

Durante el desarrollo del proyecto se implementaron diferentes conceptos vistos en el curso, permitiendo construir una aplicación organizada, modular y funcional.

## Funcionalidades implementadas

- Registro de clientes
- Gestión de servicios
- Creación y administración de reservas
- Validación de datos
- Manejo avanzado de excepciones
- Registro automático de eventos y errores mediante logs
- Simulación de operaciones válidas e inválidas
- Ejecución estable del sistema sin interrupciones

## Conceptos aplicados

- Abstracción
- Herencia
- Polimorfismo
- Encapsulación
- Manejo de excepciones personalizadas
- Uso de archivos TXT para logs
